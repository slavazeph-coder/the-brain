# Launch Checklist

## Before launch
- [ ] Replace placeholder media with a real GIF or MP4 poster image
- [ ] Set the GitHub social preview image
- [ ] Add at least 5–10 `good first issue` tickets
- [ ] Test the clone → install → run flow from scratch
- [ ] Deploy the static demo
- [ ] Update `src/constants/site.js` with the final demo URL
- [ ] Prepare X, HN, and Reddit copy in advance
- [ ] Line up a first wave of stars and shares

## Launch day
- [ ] Post Show HN and link directly to the repo
- [ ] Post the X thread with the GitHub link at the end
- [ ] Reply quickly for the first 2 hours
- [ ] Post the ML-focused Reddit version
- [ ] Post the R3F / React-focused Reddit version later
- [ ] Drop the project in neuromorphic communities
- [ ] Monitor star velocity, not just total stars
- [ ] Convert repeated feedback into GitHub issues

## Reminder
Trending is about momentum. Compress attention into one strong day.
