Crumb LLM — Wave-Field Kernel Explorer

Each attention head is replaced by a damped-cosine kernel with three learnable physics parameters. Tune them and watch how a token's information propagates across the field — the mechanism that turns O(N²) attention into O(N log N).

k(t) = exp(−α|t|) · cos(ωt + φ)

Head parameters

Dampingα 0.40
How far information travels. High α = local head, low α = global head.
Frequencyω 1.00
Which spatial patterns the head is tuned to. Like tuning a radio dial.
Phaseφ 0.00
Where in the oscillation cycle the kernel starts. Shifts the head's viewpoint.
