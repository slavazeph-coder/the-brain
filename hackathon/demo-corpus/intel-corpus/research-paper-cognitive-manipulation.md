# Brain-Region-Aware Detection of Coercive Communications: A Hybrid Architecture for Enterprise Cognitive Defense

**Synthetic research paper excerpt · Submitted to NeurIPS-Adjacent
Workshop on AI Safety · 2026**

## Abstract

We present a hybrid architecture for detecting coercive
communications in enterprise text streams. The architecture combines
a deterministic regex-based scoring layer with an on-demand large
language model (LLM) intent classifier and a brain-region routing
function that maps detected manipulation patterns to seven canonical
cortical/subcortical regions. We evaluate the architecture on a
17-sample multi-class corpus spanning phishing, coercive workplace
communications, marketing, and synthetic AI safety messages. The
hybrid system achieves an F1 of 0.81 on manipulation detection,
substantially exceeding the deterministic baseline (F1 = 0.18) while
remaining cost-bounded through escalation gating. We further
demonstrate that the brain-region routing function provides
explainability properties not available from binary classifiers and
enables triage prioritization by attack target rather than by attack
likelihood.

## 1 Introduction

Enterprise communications increasingly originate from or are
mediated by AI systems. Phishing copy is generated by large language
models. Internal communications are drafted with LLM assistance.
Customer service chatbots negotiate refund policies. The cognitive
payload of these communications — the patterns by which they shape
attention, trust, and decision-making in human readers — is
inadequately measured by current security and compliance tooling.

Existing literature on phishing detection is dominated by binary
classifiers operating over surface lexical features [3, 7, 14] or
end-to-end neural models trained on large corpora of labeled emails
[9, 22]. These approaches treat manipulation as a binary property of
the message rather than as a _cognitive interaction_ between the
message and a particular reader's neurocognitive vulnerability
profile.

## 2 Architecture

We model manipulation pressure as a function of four observable
dimensions:

- Emotional activation $E$ — fear, outrage, panic indicators
- Cognitive suppression $S$ — urgency stack, certainty theater
- Manipulation pressure $M$ — composite of $E$ and $S$
- Trust erosion $T$ — sensationalism, coercive framing

The deterministic baseline scores text via regex pattern matching
across these four dimensions, normalized as:

$$M_\text{det}(x) = \alpha \cdot E_\text{regex}(x) + \beta \cdot S_\text{regex}(x)$$

where $\alpha = \beta = 0.5$ in the default configuration.

The LLM intent classifier then computes per-intent confidence scores
$c_i \in [0, 1]$ for each intent label $i$ in our 38-class
taxonomy:

$$M_\text{intent}(x) = \sigma\left( \sum_{i \in \mathcal{I}} w_i \cdot c_i(x) \right)$$

where $\sigma$ is the logistic function and $w_i$ are per-intent
weights derived from a small calibration set.

The combined manipulation pressure is taken as the maximum of the
deterministic and intent-augmented scores:

$$M_\text{combined}(x) = \max(M_\text{det}(x), M_\text{det}(x) + \Delta_\text{intent}(x))$$

where $\Delta_\text{intent}$ is the intent classifier's reported
delta over the deterministic baseline.

## 3 Brain-region routing

We map detected manipulation patterns onto seven canonical brain
regions: amygdala (AMG), basal ganglia (BG), thalamus (THL),
prefrontal cortex (PFC), hippocampus (HPC), cortex (CTX), and
cerebellum (CBL). Each intent label has a primary region target
derived from the cognitive neuroscience literature on social
influence [8, 16, 21]. The routing function returns the dominant
region by weighted vote of the active intent labels.

This routing serves three purposes. First, it provides an
_explainable_ output: a security analyst can read "amygdala
dominant" as "this attack targets fear processing." Second, it
provides a _triage signal_: attacks targeting suppression vectors
(PFC dampening, BG action gating) are more dangerous to time-pressed
operators than attacks targeting fear (which trained operators
recognize). Third, it enables _persona-aware scoring_: the same
content can be scored under different reader vulnerability profiles
by reweighting region sensitivity.

## 4 Evaluation

We evaluated the hybrid architecture on a 17-sample corpus spanning
six attack classes. The deterministic baseline achieved F1 = 0.18,
catching three of seventeen samples above 0.30 manipulation
pressure. Adding the intent classifier raised F1 to 0.81 and
recovered all four sophisticated phishing samples (CEO BEC,
MFA-fatigue, vendor invoice, recruiter trojan) that were invisible
to the regex layer. Two false-positive cases produced by the
deterministic layer (one medical pre-op briefing flagged as
hidden-truth conspiracy; one legitimate autonomous-vehicle takeover
prompt flagged as manipulation) were correctly cleared by the
intent classifier with appropriate annotations.

Per-corpus results, scoring methodology, and complete intent label
taxonomy are available in the supplementary materials.

## 5 Discussion

The brain-region routing function provides a form of explainability
that binary classifiers cannot. Production deployments of this
architecture report that security analysts triage attacks targeting
PFC suppression and BG action-gating with greater priority than
attacks targeting AMG fear, because trained operators are more
resilient to the latter. The persona-aware scoring extension enables
red-team teams to identify which roles within an organization are
most vulnerable to a given attack class — a capability not available
from corpus-level F1 metrics.
